# Changelog

All notable changes to this template are documented here.
This project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-01-01

Initial release of the **free teaser edition**.

### Added
- Astro 5 + Tailwind CSS 4 project with fully static output.
- Landing page: hero, features, testimonials, FAQ, and CTA sections.
- Styled 404 page.
- Dark/light mode with no flash of the wrong theme, responsive layout, and
  accessibility basics (landmarks, skip link, focus states, reduced-motion).
- Single-file theming via `src/config/site.ts` (brand colour, font, logo,
  identity) plus `src/config/content.ts` for marketing copy.

### Not included (see the paid EU-Ready SaaS Landing Kit)
- Pricing page, Markdown blog, about/contact pages.
- GDPR legal skeletons (privacy, terms, cookie policy, imprint).
- Cookie consent banner with script gating.
